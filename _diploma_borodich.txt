РАЗРАБОТКА ИНТЕЛЛЕКТУАЛЬНОЙ СИСТЕМЫ МОНИТОРИНГА ПСИХОЭМОЦИОНАЛЬНОГО СОСТОЯНИЯ С АДАПТИВНЫМ ПРОГНОЗИРОВАНИЕМП.В. Травничева, А.Д. БородичВитебский государственный университет имени П.М. Машерова,г. Витебск, БеларусьАннотация: В работе представлена интеллектуальная система мониторинга психоэмоционального состояния, основанная на интеграции гетерогенных данных и адаптивных алгоритмах машинного обучения. Разработано кроссплатформенное мобильное приложение, осуществляющее комплексный сбор данных с носимых устройств, анализ текстового контента и ведение цифрового дневника настроения. Особенностью системы является способность к динамической адаптации моделей прогнозирования под индивидуальные характеристики пользователя. Экспериментальные результаты демонстрируют повышение точности прогнозирования стрессовых состояний на 27% и увеличение пользовательской вовлеченности на 23% по сравнению с базовыми решениями.Ключевые слова: интеллектуальный мониторинг, психоэмоциональное состояние, машинное обучение, адаптивные алгоритмы, кроссплатформенная разработка, прогнозирование стресса.DEVELOPMENT OF AN INTELLIGENT MONITORING SYSTEM FOR PSYCHOEMOTIONAL STATE WITH ADAPTIVE PREDICTIONP.V. Travnicheva, A.D. BorodichVitebsk State University named after P.M. Masherov,Vitebsk, BelarusAbstract: *This paper presents an intelligent monitoring system for psychoemotional state based on integration of heterogeneous data and adaptive machine learning algorithms. A cross-platform mobile application has been developed that performs comprehensive data collection from wearable devices, text content analysis and digital mood diary maintenance. The system's distinctive feature is its ability to dynamically adapt prediction models to individual user characteristics. Experimental results demonstrate a 27% improvement in stress state prediction accuracy and a 23% increase in user engagement compared to baseline solutions.*Keywords: intelligent monitoring, psychoemotional state, machine learning, adaptive algorithms, cross-platform development, stress prediction.Введение. Современные требования к мониторингу психического здоровья диктуют необходимость разработки интеллектуальных систем, способных осуществлять превентивный контроль состояния пользователей [1]. Большинство существующих решений ограничиваются сбором данных из отдельных источников и не обеспечивают персонализацию рекомендаций [2]. В связи с этим актуальной задачей является создание систем, способных интегрировать разнородные данные и адаптировать алгоритмы анализа под индивидуальные особенности пользователей [3].Цель работы — разработка интеллектуальной системы мониторинга психоэмоционального состояния с адаптивным прогнозированием на основе комплексного анализа данных из multiple источников.Основной текст. Материал исследования включает данные о физической активности, метрики сна с носимых устройств, текстовый контент из социальных сетей и субъективные оценки настроения. В работе использованы методы машинного обучения для прогнозирования стрессовых состояний и формирования персонализированных рекомендаций. Технологический стек реализации включает: Flutter / Dart для кроссплатформенной разработки, Python / FastAPI для серверной части и ML-алгоритмов, PostgreSQL для хранения данных и результатов анализа.Архитектура системы предусматривает трехэтапную обработку данных: агрегацию из разнородных источников, извлечение признаков и классификацию с прогнозированием. Разработанный алгоритм динамически адаптируется под изменения в поведенческих паттернах пользователя.Интерфейс приложения включает следующие ключевые модули: дашборд с визуализацией индекса благополучия, журнал для ежедневного отслеживания настроения, систему персонализированных рекомендаций, модуль превентивных уведомлений, детальную статистику и аналитику.Экспериментальная оценка эффективности системы проводилась на фокус-группе в течение 4 недель. Результаты показали, что точность прогнозирования стрессовых состояний в экспериментальной группе превысила аналогичный показатель контрольной группы на 27%. Пользовательская вовлеченность, измеряемая по частоте взаимодействия с приложением, оказалась на 23% выше.Заключение. Разработанная система представляет собой практическую реализацию подхода к интеллектуальному мониторингу психоэмоционального состояния. Использование адаптивных алгоритмов машинного обучения позволило существенно повысить точность прогнозирования и персонализацию рекомендаций. Полученные результаты подтверждают перспективность предложенного подхода для создания эффективных решений в области ментального здоровья.ЛитератураСмирнов А.В.
